Does Protractor collect any data?
Protractor requests permissions to the page only to place the tool on the page. Protractor does not collect or store any data at all.
How can I use Protractor on JPGs or PDFs?
You need to enable offline file access for this extension in the Chrome settings. Here's how:
  • Enter chrome://extensions in the address bar
  • Find Protractor, and click Details
  • Toggle Allow access to file URLs
Why do the angles increase backwards?
You can choose clockwise or counterclockwise rotation in the options:
  • Enter chrome://extensions in the address bar
  • Find Protractor, and click Details
  • Find and click Extension Options
  • Toggle Rotation Direction
Why can't I see the Protractor?
Protractor will not work inside the Chrome web store (where it's installed from). You must be on a page outside of the store.
Protractor will not work on any URL starting with chrome-extension://.
If you are in Incognito mode, Protractor must be granted permission:
  • Enter chrome://extensions in the address bar
  • Find Protractor, and click Details
  • Toggle Allow in incognito
How is Protractor licensed for distribution?
Many schools and government agencies use Protractor. It's licensed under the MIT license, which is an extremely permissive distribution license.
The code is also fully open source and accessible here: https://github.com/ben-burlingham/protractor.
